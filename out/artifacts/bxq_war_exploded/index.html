西安电子科技大学导览
图书馆

西安电子科技大学图书馆的前身于1949年8月1日在张家口由中央军委工校创建。现已逐步形成以信息和电子学科为特 色，理、工、文、管、经等多学科文献并存的文献资源保障体系。详情 >>

主楼

主教学楼由中国人民解放军总后勤部批准，北京军区后勤部设计处设计，西安市第二建筑公司承建。1956年11月21日 开工，1958年12月21日竣工。俯视望去，主教学楼似一架巨大的飞机。详情 >>

办公楼

办公楼由陕西省第一建筑设计院设计，陕西省第十建筑工程公司承建。办公楼呈“工”字形，前楼为四层， 腰部为三层，后楼为六层的书库。办公楼2002年由原用途的图书馆改造而成。 详情 >>

科技楼

科技楼92年7月6日开工，95年1月18日竣工。科技大楼为地上11层并为一层地下室。1995年被建筑工程质量监督 站评为市建筑质量优质样板工程。科技大楼用于科学研究及实验用房。 详情 >>

科技实验楼

科技实验楼总面积27859平方米，为“T”型建筑，主楼18层，辅楼3层，在第3层设有与科技楼连接的长廊。该楼中主要 有智能信息处理科学与技术创新引智基地和生命科学技术学院等。详情 >>

西大楼

西大楼1960年5月28日开工，1963年10月29日竣工，设计标准是可容纳2500名学生同时上课教学楼，大楼呈“工”字型。 西教学楼主要用学校教学、科研和部分学院及教师的日常办公用房。详情 >>

东大楼

东大楼1961年11月开工，1965年12月12日竣工。该楼原设计楼层为五层，因当时正逢国家遭受自然灾害，工程在建 中一度下马，后又开工，但修了一面。东大楼主要用于学校的日常教学。详情  >>

体育运动中心

习惯称之为“风雨操场”，由深圳奥托玛大跨度自动建造工程公司设计，陕西省宝天建筑公司承建，建筑总面积2300平 方米，其中用特殊材料覆盖、36米大跨度屋顶是这一建筑的特点。详情 >>

文化活动中心

文化活动中心由电子工业部批准，电子工业部第设计院设计，陕西省省建三公司承建。文化活动中心用于学校开展集会、 娱乐等活动，后面的贴房为科研用房和学校的电话总机用房。详情 >>
